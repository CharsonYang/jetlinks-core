# 设备定义

设备主要由3部分组成：
1. 属性，对设备的描述，如： 型号，当前电量。
2. 功能，对设备的操作，如： 打开开关，获取设备状态。
3. 事件，设备主动上报数据，如：定时上报温度，传感器触发警报。
